#Hinglish Sentiment Analysis

Setup Instructions

1. Clone the repository:
   git clone https://github.com/RevatiKeskar/Hinglish-Sentiment-Analysis
   cd <repository-folder>

2. Create a model folder in the root directory of the project.

3. Download the model from https://drive.google.com/drive/folders/1KBZ_8zSIb_bj4nDsZZF1q1abn-UqkhJd?usp=sharing and place it inside the model folder.

4. Run the project:
   python app.py
   

Notes
- Make sure the model folder contains the correct model file.
